# practice-nuxt3-namelist-api(名前リスト作成アプリ)

- 学習内容：Nuxt3 + Laravel API 連携
- 作成時期：2025年12月
- 練習テーマ：CRUD

# 使用方法

・名前の追加
入力フォームのテキストボックスに文字列を入力します。「追加」ボタンを押すと、名前リストに追加されます。

・名前の更新
名前リストの名前をクリックすると、編集することができます。「更新」ボタンを押すと、アラート出力後に該当行の名前を更新することができます。

・名前の削除
名前リストの「削除」ボタンを押すと、アラート出力後に該当行の名前を削除することができます。
